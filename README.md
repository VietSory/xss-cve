# XSS-CVE
Phát triển một module (payload/exploit/scanning/….) cho Metasploit. Mô phỏng đối với: XSS.


Đây là module khai thác xxs dựa trên CVE-2020-29606 – Stored Cross-Site Scripting (XSS) trên Pluck CMS 4.7.13,bạn có thể dựng nó bằng 2 file docker tôi đã cung cấp.


Lưu các module metasploit ở link trong file tôi đã để sẵn.


Tôi cũng dựng 1 docker đơn giản để demo 3 loại xss cơ bản.
